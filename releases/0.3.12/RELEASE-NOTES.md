# PersonalVABot 0.3.12 — Windows Desktop Beta

## Release status

This is a validated Windows Desktop Beta release, not a final `1.0.0` production release.

## Highlights

- multi-client workspaces;
- client memory and preferences;
- project and task lifecycles;
- attendance and DTR workflows;
- billing agreements and earnings references;
- document and invoice-support workflows;
- recurring tasks and reminders;
- 72 reusable templates;
- smart recommended next actions;
- activity and health monitoring;
- local SQLite storage;
- backup, restore, and export workflows;
- credential-safe diagnostics;
- installer and restart-persistence validation.

## Important notes

- The installer may be unsigned and can trigger Windows SmartScreen.
- Download only from the official GitHub Release.
- Verify the published SHA-256 checksum.
- Create a fresh backup before uninstalling, upgrading, or performing manual cleanup.
- Stable local data is stored under `%APPDATA%\PersonalVABot`.

## Known scope

This release is Windows-only. Web, Slack, multi-tenant, payroll, payslip, manager portal, and client portal capabilities remain future plans unless announced separately.
