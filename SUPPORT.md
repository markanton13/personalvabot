# Support

PersonalVABot is currently a Windows Desktop Beta.

Before requesting support:

1. confirm the installed version;
2. create a fresh backup;
3. review the installation, privacy, and validation documents;
4. remove tokens, client data, and private records from screenshots and logs;
5. describe the expected behavior and actual behavior;
6. include safe reproduction steps.
