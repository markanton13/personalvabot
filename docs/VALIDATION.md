# Windows Desktop Beta Validation

## Validated release

```text
PersonalVABot 0.3.12 — Windows Desktop Beta
```

## Tested areas

### Installation and launch

- installer launch;
- desktop application startup;
- local configuration;
- restart persistence;
- stable runtime data location.

### Client and project workflows

- create multiple clients;
- preserve separate client memory;
- create and manage projects;
- link operational records to the correct client.

### Task lifecycle

- capture tasks;
- move tasks through Inbox, Active, Blocked, Review, Done, and Archived;
- reopen archived or completed work;
- protect permanent deletion with explicit confirmation;
- preserve records after application restart.

### Attendance and DTR

- time-in and time-out;
- break handling;
- expected and actual hours;
- attendance status;
- editing and corrective workflows.

### Billing and documents

- billing agreements;
- earnings references;
- document records;
- document lifecycle;
- invoice-support workflows.

### Recurring work and templates

- recurring tasks;
- reminders;
- reusable templates;
- smart recommended next actions.

### Reliability and recovery

- backup creation;
- backup visibility;
- restore behavior;
- data persistence;
- credential-safe diagnostics;
- activity and health monitoring.

## Beta limitations

- Windows only;
- installer may be unsigned;
- trusted beta testing remains in progress;
- future web, Slack, multi-tenant, payroll, payslip, manager portal, and client portal functionality is not part of this release.
